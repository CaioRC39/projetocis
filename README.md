# ULA de 4 bits com Carry Look-Ahead (CLA)
## *Projeto individual – Disciplina: Projeto de Circuitos Integrados*

- Projeto e implementação manual de uma Unidade Lógica e Aritmética (ULA) de 4 bits com Carry Look-Ahead (CLA) no software Microwind, extrapolando os requisitos originais da disciplina para aprofundamento em arquitetura de hardware digital.

- Otimização de tempo de propagação, com definição manual das dimensões dos transistores (tamanho de gate, n-well e demais parâmetros tecnológicos), visando melhoria de desempenho em relação a somadores convencionais.

- Desenvolvimento focado em nível físico (layout), aplicando conceitos de projeto VLSI, temporização e eficiência de circuitos digitais, com validação funcional da ULA para operações aritméticas e lógicas.
